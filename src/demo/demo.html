jquery-rsButton plugin demo

A CSS button that displays several frames when the user clicks on it.

Every push button has two states: the unpressed and the pressed state. While some sites only display a unique unpressed state image of a button, others go a bit further and show a different image for the pressed state, giving a nice visual feedback when a button is pushed down.

This plugin makes this visual feedback even more realistic, by quickly showing a series of frames between these two states.

These buttons are made out of pure CSS, no images whatsoever.

They resize to fit any content

If your CSS is based on em units (as it should always be, even for fixed size layouts), then they can fit any screen size.
